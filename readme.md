### **问题是：**             
在远离输入端口的输出端口位置接大电流(***>200mA***)负载时，              
靠近输入端的端口就算**没接负载**也会有大概10~24mA被检测，              
在飞线测试时，发现每个输出端口的供电输入单独一根线即可避免。       
虽然现象可能已经解决，
但理论分析可能需要补习关于***电源完整性(PI)***的内容。
